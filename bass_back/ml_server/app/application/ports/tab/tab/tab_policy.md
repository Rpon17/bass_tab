2. 생성 이론

가중치가 높을수록 우선순위
-1. 기본적으로 음이 있는 피치마다 가중치 10을 둔다
-2. string을 바꾼다 -> -5
-3. 프랫을 1개 바꾼다 -> -1 하지만 같은줄에서 프랫을 7개이상 바꿀때마다 가중치가 -3씩 추가된다
-4. 0번줄을 친다 -> 가중치 그대로
-5. 네 마디마다의 평균을 내서 그 프랫의 평균 피치를 찾는다 여기서 40 45 50 55같은 피치는 제외한다 0 -> 악보상 한줄
    그리고 그 네마디의 피치 평균이
    28~40 -> 4번줄
    40~52 -> 3번줄
    52~64 -> 2번줄
    64이상 -> 1번줄로 하는데
    곡의 첫 마디를 제외하고는 이 범위에대해 가중치를 +6 한다

-6. 만약에 string_cut_t만큼의 박 이내에 string을 2개 이상의 string을왔다갔다하면 이건 규제함
-7. 그리고 sufficient_t라면 최대한 0프렛에 가깝게 하지만 가중치에서 너무 벗어나지 않게
